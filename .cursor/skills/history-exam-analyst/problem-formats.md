# 문제 형식 분류 가이드

## 1차 분류 (문제 형식)

1차 형식은 **상호배타**입니다. 한 문항에 여러 형식이 겹치면 아래 **우선순위**에서 먼저 매칭되는 하나만 선택합니다.

| 우선순위 | 형식 | 판별 기준 |
|----------|------|-----------|
| 1 | 역사 신문형 | `○○신문` **지면** 레이아웃이 분명할 때 |
| 2 | 영상자료형 | TV·다큐·드라마·극장 뉴스 등 **영상·방송 프레임** (기획·계획 문서만 있으면 해당 없음) |
| 3 | 퀴즈형 | 골든벨·퀴즈 쇼 UI |
| 4 | 마인드맵형 | 마인드맵·개념도 |
| 5 | 카드형 | 카드·카드뉴스·QR 등 **카드 레이아웃**이 주된 제시 장치 (유물 여부 무관) |
| 6 | 인터뷰형 | 기자(또는 명시적 인터뷰어) ↔ 역사 인물 |
| 7 | 답사형 | **야외** 유적지를 **가이드가 소개** |
| 8 | 실내전시형 | **실내** 전시·진열장·도슨트·키오스크 등 **유물·문화유산** 제시 |
| 9 | 메신저형 | 모바일 메신저·채팅 UI가 주된 제시 장치 |
| 10 | 챗봇형 | AI·인공지능 챗봇 대화 UI가 주된 제시 장치 |
| 11 | 수업형 | **현대 교실·학교 활동의 시각 프레임**이 주(교복·교사–학생 학습 장면, 교실·전자칠판, 모둠·역할극·발표 **장면** 등) |
| 12 | 대화형 | **수업·메신저·챗봇이 아닌** 인물들 사이 말풍선 대화만 (토론회·역사 사건 삽화가 주가 아닐 때) |
| 13 | 계획서형 | 답사·수행평가·다큐·웹소설 등 **텍스트 중심 계획서·기획서·기획안** |
| 14 | 역사 장면형 | 역사적 사건·상황·인물을 **시대 배경 삽화·장면**으로 제시 (교실 프레임·토론회 프레임 아님) |
| 15 | 역사 토론형 | 역사 인물·가상 인물의 **토론회·대립 토론** 형식이 주된 제시 |
| 16 | 자료 제시형 | **잔여** (문헌·사진·보고서·Q&A·백과사전·게시판 그래픽 등) |

### 답사형 vs 실내전시형

| | 답사형 | 실내전시형 |
|--|--------|------------|
| 장소 | 야외 유적지 | 실내(박물관·미술관·전시관 등) |
| 제시 | 가이드가 현장에서 유적 소개 | 유물·문화유산을 실내 자료로 제시 |
| 예 | 고인돌·궁궐·표지석 앞 가이드 | 진열장 유물, 도슨트, 전시 해설, 키오스크 |

### 역사 장면형 vs 역사 토론형 vs 대화형

| | 역사 장면형 | 역사 토론형 | 대화형 |
|--|-------------|-------------|--------|
| 주된 장치 | 역사 사건·상황 **삽화·장면** | **토론회·대립 토론** 프레임 | 말풍선 **대화**만 |
| 예 | 우정총국 앞 개화파, 전투·봉기 장면 | 가상 토론회, 역사 인물 토론회(연단) | 학생·왕과 신하 등 단순 대화 |
| 비고 | 말풍선이 있어도 사건 장면이 주이면 장면형 | 토론회 제목·연단·대립 구도가 주 | 토론회·사건 삽화가 주가 아닐 때 |

### 경계 한 줄 규칙

- 가이드 **독백** + 야외 유적 → **답사형** (대화형 아님)
- 실내 진열/전시 + 도슨트·오디오 → **실내전시형** (답사형 아님)
- 야외 유적 **사진만** 화면·교실에 제시 → **자료 제시형**
- TV·뉴스 **방송 화면** → **영상자료형** / `○○신문` **지면** → **역사 신문형**
- 카드 UI가 본문 중심 → **카드형** (문화유산 카드·유물 카드 포함)
- 메신저·채팅 UI가 주 → **메신저형** (교사–학생이어도 메신저 UI면 메신저형)
- AI·챗봇 UI가 주 → **챗봇형**
- **현대** 교실·교복·교사–학생 학습·모둠·역할극·발표 **장면**이 주 → **수업형** (대화형 아님; 메신저·챗봇 UI가 아니면)
- 문항 줄기·제목의 “수업·발표·토론”만으로 수업형 **금지** (제시 자료의 시각 구조로 판별)
- 역사 사건·상황 **삽화·장면**이 주 → **역사 장면형** (수업형·자료 제시형 아님)
- `가상 토론회`·역사 인물 **토론회**(연단·대립) → **역사 토론형**
- 게시판·메모·노트·탐구 안내문 **그래픽만** (교실·역사 장면 없음) → **자료 제시형**
- TV·뉴스 **방송 화면** → **영상자료형** (수업형 금지)
- 수업·메신저·챗봇·역사 장면·역사 토론이 아닌 단순 인물 대화 → **대화형**
- 답사·수행평가·다큐·웹소설 등 **계획서·기획서·기획안** 본문 → **계획서형** (완성 보고서·결과물은 자료 제시형 또는 수업형)
- 기자 인터뷰가 주 → **인터뷰형** (학생끼리 Q&A·게시판은 자료 제시형)

## 2차 분류 (세부 형식) 예시

### 자료 제시형
- 단일 자료
- 복합 자료 (2개 이상)
- 자료 + 지도/도표
- 백과사전
- Q&A
- 조서·법령
- 유적·역사 사진
- 게시판·메모·노트·탐구 안내 그래픽
- 수행평가·탐구 **보고서**(결과물)

### 역사 장면형
- 역사 장면 삽화
- 전투·봉기·사건 장면
- 인물·상황 삽화 (토론회 프레임 아님)
- 시대 배경 재현 장면

### 역사 토론형
- 가상 토론회
- 역사 인물 토론회
- 대립 토론(연단·마이크 등)

### 카드형
- 역사 인물 카드
- 인물 카드
- 카드뉴스
- QR코드 인물 설명
- 문화유산 카드
- 역사 용어 카드

### 영상자료형
- 오늘의 역사
- TV 영상 자료
- TV 뉴스·뉴스 방송
- 다큐멘터리 **예고·장면** (기획 문서만이면 계획서형)
- 드라마 장면 자료
- 극장 뉴스 영상
- 미디어 자료

### 메신저형
- 모바일 메신저 대화
- 채팅·메신저 UI

### 챗봇형
- 인공지능 챗봇 대화
- AI 질의응답 UI

### 수업형
- 토론 수업 (교실·모둠 **장면**이 보일 때)
- 발표 수업 (교복·전자칠판 등 **발표 장면**이 보일 때)
- 교사와 학생 대화
- 역할극 (수업 장면이 주)
- 모둠 활동 (교실·모둠 **장면**이 보일 때)
- 교사 설명(수업 장면이 주)
- (뉴스 보도·게시판 그래픽·역사 삽화만 있는 경우는 해당 없음)

### 대화형
- 학생 2인 대화
- 학생 3인 대화
- 역사 인물 대화
- 왕과 신하 대화
- (수업·메신저·챗봇·역사 장면·역사 토론이 주가 아닐 때)

### 계획서형
- 답사 계획서
- 수행평가 계획서
- 다큐멘터리 기획서·기획안
- 웹소설 기획서
- 인물 탐구 계획서(문서가 주이고 수업 장면이 아닐 때)

### 인터뷰형
- 역사 인물 인터뷰
- 전문가 인터뷰

### 역사 신문형
- 단일 기사
- 복수 기사
- (신문 지면이 아닌 TV·방송은 영상자료형)

### 퀴즈형
- 골든벨
- OX 퀴즈

### 답사형
- 가이드 1명 설명 (야외)
- 가이드 2인 대화 (야외)
- 유적 답사 (야외 가이드)

### 실내전시형
- 박물관 유물 관람·해설
- 미술관·전시 해설
- 박물관 키오스크·전시 안내
- 실내 진열장·홀로그램 전시
- 유물 제시 (실내)

### 마인드맵형
- 개념 연결
- 인과 관계도

## 신규 형식 추가 절차

1. 기존 1차 분류로 분류 불가능한지 확인
2. 사용자에게 새 형식명 제안 및 승인 요청
3. 승인 후 이 문서에 예시·우선순위 위치와 함께 추가
4. CSV에 해당 형식명으로 기록
